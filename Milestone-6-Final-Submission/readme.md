## To run locally
### Setup:

- On the terminal change the directory to `Backend` run the following command

```
sh local_setup.sh
```

```
python3 app.py
```

- Open a new tab and run
```
mailhog
```
- Open a new tab On the terminal change the directory to `Frontend` run the following command

```
npm install
```
```
npm run serve
```
